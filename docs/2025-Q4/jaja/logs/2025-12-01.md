### MySQL 실행
```bash
mysql -uroot -p --host=localhost --socket=/tmp/mysql.sock
mysql -uroot -p --host=127.0.0.1 --port=3306
mysql -uroot -p
```

- `--host=localhost` 옵션을 사용하면 소켓 파일을 통해 접속
    - Unix domain socket을 이용하는 방식(IPC; Inter Process Communication)
- `--host=127.0.0.1`을 사용하는 경우는 loopback ip지만 TCP/IP 통신 방식 사용
- 호스트와 포트를 명시하지 않으면 localhost로 소켓 파일 접속
