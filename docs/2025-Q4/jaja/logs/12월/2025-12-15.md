# 12-15-slow-query-log

Real MySQL 8.0 1 - 아키텍처

---

### 슬로우 쿼리 로그

- 서비스에서 사용되는 쿼리 중에서 어떤 쿼리가 문제인지 판단하는 데 도움
- `long_query_time` 시스템 변수에 설정한 시간 이상 소요된 쿼리 기록
- 테이블 혹은 디스크 파일로 저장
- Time(종료된 시점), 쿼리를 실행한 사용자 계정, 실행 시간, 잠금 대기 시간, 접근한 레코드와 전송한 처리 결과 수 확인
- 슬로우 쿼리 통계
- 실행 빈도 및 누적 실행 시간순 랭킹
- 쿼리별 실행 횟수 및 누적 실행 시간 상세 정보
