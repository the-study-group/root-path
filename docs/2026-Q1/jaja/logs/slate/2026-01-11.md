# Slate 생성

사이드 프로젝트 `Slate`

*created: 2026.1.11*

---

### 개발 환경 구성
- JDK *Java Development Kit*
    - = JRE *Java Runtime Environment* - 자바 실행 환경
    - 자바를 가지고 개발하는데 필요한 여러가지를 모아둔 것
    - JVM, 표준 라이브러리, Javac(자바 컴파일러), Tool(디버깅, 파일 확인, 프로세스 동작 확인)
- IDE *Integrated Development Environment*
    - 통합 개발 환경
- PostgreSQL
- Docker

### 프로젝트 생성
- Spring Boot 4.0.1
- Java 21
- PostgreSQL
- Flyway
- Spring Data JPA
- Docker

### PostgreSQL 선택 이유
- 여러 사람이 동시에 읽거나 쓰는 작업이 많음
- MySQL 대비 복잡한 join과 subquery 성능 우수
- JSON 타입 지원으로 유연한 데이터 구조 설계 가능

### MySQL vs PostgreSQL
- MySQL: 효율적인 공간 관리와 빠른 단순 읽기에 유리
- PostgreSQL: 높은 동시성과 복잡한 쿼리 성능
